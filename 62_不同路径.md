# 62. 不同路径

20200706

> 1. 一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。
>
>    机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。
>
>    问总共有多少条不同的路径？
>
>     
>
> 示例 1:
>
> 输入: m = 3, n = 2
>输出: 3
> 解释:
>从左上角开始，总共有 3 条路径可以到达右下角。
> 
> 1. 向右 -> 向右 -> 向下
>   2. 向右 -> 向下 -> 向右
>   3. 向下 -> 向右 -> 向右
>   
> 示例 2
> 
> 输入: m = 7, n = 3
> 输出: 28

题解：

```C++
class Solution {
public:
    int uniquePaths(int m, int n) {
        int dp[2][n];
        dp[0][0]=1;
        for(int i=0;i<m;i++){
            for(int j=0;j<n;j++){
                if(i!=0 || j!=0)
                    dp[i%2][j]=0;
                if(i!=0){
                    dp[i%2][j]+=dp[i%2==0?(i%2+1):(i%2-1)][j];
                    //cout<<dp[i%2==0?(i%2+1):(i%2-1)][j]<<' ';
                }
                if(j!=0){
                    dp[i%2][j]+=dp[i%2][j-1];
                    //cout<<dp[i%2][j-1]<<'J';
                }
                cout<<dp[i%2][j];
            }
        }
        return dp[(m-1)%2][n-1];
    }
};
```
